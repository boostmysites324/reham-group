# How to Get Partner Logos

## Emirates Logo
1. Go to: https://www.emirates.com/media-centre/
2. Or search: "Emirates logo PNG transparent" on Google Images
3. Download a high-resolution PNG with transparent background
4. Save as: `src/assets/emirates-logo.png` (replace existing file)

## Hyundai Logo  
1. Go to: https://www.hyundai.com/worldwide/en/company/newsroom
2. Or search: "Hyundai logo PNG transparent" on Google Images
3. Download a high-resolution PNG with transparent background
4. Save as: `public/lovable-uploads/hyundai-logo.png`

## Recommended Sources:
- Official company media centers
- Brandfolder.com
- Logos-world.net
- CleanPNG.com
- FreePNGLogos.com

Make sure logos are:
- PNG format with transparent background
- High resolution (at least 500px width)
- Official/current versions
